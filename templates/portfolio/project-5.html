Telegram Bot for Orientirungskurs exam

About Project

Telegram bot helps Ukrainians at integration courses in Germany better prepare for the “Life in Germany” exam. The bot accepts a number from 1 to 300 as input and sends the original TELS test with a hidden answer.

The code is written according to the OOP paradigm. I used the telebot library here and created the database using SQLite. I also previously wrote a parser to extract data about answers and questions from a PDF file.
